Web Declutterer

Removes all the annoying ads and just gives the actual content present in the site


# Features
- AI Summary
- Save Articles
- Word count And approx reading time 

# Tech Stack
- HTML, CSS, JavaScript
- Node.js("Backend Not implemented yet")

# How to Run
Install the live server in VS Code
Install the css,js and html files from the repo
